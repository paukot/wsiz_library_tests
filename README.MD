# Requirments

- Python 3.13+

## Installation process

```shell
  pip install -r requirements.txt
```

Copy .env.example to .env and fill login and password, chromedrive path is optional if your instance is in default path

## Usage

To run tests

```shell
  pytest
```

To run test only in a file

```shell
  pytest tests/test_library_user_profile.py
```
